# animalsounds

``` r
library(animalsounds)
```

Print what an animal says:

``` r
animal_sounds("dog", "woof")
#> [1] "The dog goes woof!"
animal_sounds("giraffe")
#> [1] "The giraffe makes no sound."
```

Also contains data:

``` r
farm_animals
#>   animal sound
#> 1    cow   moo
#> 2  horse neigh
#> 3   duck quack
```
